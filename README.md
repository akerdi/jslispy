# jslispy

使用js 覆写[clispy](https://github.com/shaohung001/buildyourownlisp)

## Language

Typescript

## 开发

    cp .vscode/launch.example.json .vscode/launch.json // 然后按键盘 F5
或者

    yarn dev

## 运行

    yarn build && node dist/main.js
